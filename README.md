# Installation

## Composer:
```bash
composer require achse/nette-money-input
```

## Javascript Dependencies
```
npm install jquery
```

If you are using webloader:
```neon
webloader:
	js:
		default:
			files:
				- %wwwDir%/../vendor/achse/nette-money-input/assets/moneyInput.js
```
